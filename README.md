## How to run

To run application you must clone repo and use this commands:

1. `npm install`
2. `npm run dev`

## Technologies

- Vue
- Typescript
- TailwindCSS
- .NET 7 (planned to move to version 8 on release)
- PostgreSQL
- SignalR
